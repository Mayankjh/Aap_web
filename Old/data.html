Aam Aadmi Party – Data Analysis Cell

Curriculum

During the course of the internship, data analysts would be processing data sets in order to draw conclusions about the information they contain, with the aid of specialized systems and software. The interns working on and analysing various data sets and refining them on Python. They’d be made adept at using data analytics technologies and techniques which are widely used in commercial industries to enable organizations to make more-informed business decisions. It would also include gaining a comprehensive understanding of Python and building machine learning algorithms for policy making and for electoral usage. The interns would be trained to use data analytics as a tool to increase revenues, improve operational efficiency, optimize marketing campaigns and respond more quickly to emerging market trends and gain a competitive edge over rivals.

Total working Days = 44

Total Modules =9 excluding Assessment task

Module 1 -10 Days

Extraction of Data

Website Crawling using python code and Google APIs

  • Write Python code to download all big data files in one command
  • Write Python code to convert into image file.
  • Write Python code to crop text part
  • Convert that file into text using Google API.
OR

To make neural network model to read hindi , gujrati , urdu text files

Module 2 -7 Days

Converting Into Excel and cleaning of Data:

  • Write Regular expression for converting that text into excel and mapping of text to respective columns and rows of excel file
  • Cleaning of Data using python code by finding various patterns

Module 3 -5 Days

Data Mining

  • Finding standard deviation , mean , mode , maxima , minima
  • a) among booths ,
  • b) among parties ,
  • c) among years

Module 4 -3 Days

Caste Analysis

  • to find percentage of every caste in every polling station
  • to make Bot for same .

Module 5 -3 Days

Age Analysis

  • to find percentage of various age groups for every polling station li>

Module 6 -7 Days

Heat Maps

  • to make Heat Maps showing % of caste , class , age on vidhansabha , district and state maps using python/R code or Tableau
  • to make heat maps showing vote % of every party for every year in various assembly and parliamentary constituencies

Module 7 -5 Days

Graphs , pi-charts , correlation Matrix

  • to make graphs showing various variations such as vote within years among parties for every AC etc etc

Module 8 -3 Days

Prediction, conclusion/story writing

Module 9 -1 Day
